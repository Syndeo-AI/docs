---
title: "Quickstart"
description: "Install the CLI and register your first cluster."
---

## 1. Install the CLI

```bash
brew tap Syndeo-AI/tap
brew install resyndeo
```

## 2. Get an install token

Sign in at [app.resyndeo.com](https://app.resyndeo.com) and copy a fresh install token.

## 3. Install the cluster

```bash
resyndeo install --token <token>
```

## 4. Wait for registration

The CLI provisions a local k3d cluster and registers it with the Resyndeo backend.

## 5. Verify

Your cluster appears under **Clusters** at [app.resyndeo.com](https://app.resyndeo.com).
