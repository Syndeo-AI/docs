---
title: "Install a plugin"
description: "Deploy a plugin from the marketplace onto your cluster."
---

Plugins are Helm charts the Resyndeo backend ships to your cluster via the in-cluster agent. You install one from the marketplace in the dashboard — no `kubectl` or `helm` required.

## Prerequisites

- A cluster registered with Resyndeo. See [Install a cluster](/install-cluster).
- Cluster online (the in-cluster agent must be connected to the backend).

## Steps

1. Sign in at [app.resyndeo.com](https://app.resyndeo.com).
2. Open **Marketplace**.
3. Find the plugin card and click **Deploy**.
4. If the plugin has settings (e.g. Postgres), fill them in and confirm.
5. The card flips **Deploy → Pending… → Installed** once the chart lands on your cluster.

## Verify

The card shows **Installed**, and on a reload it renders **Installed** directly. The release also appears in your cluster:

```bash
kubectl get pods -A | grep <plugin-name>
```

## Troubleshooting

- **Stuck on Pending.** The agent is offline or the install failed. Reload the page — the card returns to **Deploy** so you can retry. Check that your cluster is running (`docker ps`) and that `resyndeo` is connected.
- **No Deploy button.** Your account isn't linked to a cluster yet. Run [`resyndeo install`](/install-cluster).

See [Plugins](/plugins) for what each plugin does.
