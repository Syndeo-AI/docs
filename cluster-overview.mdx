The cluster overview replaces the install confirmation once your cluster is registered. It's the default view for a cluster in the dashboard.

## Where it lives

Sign in at [app.resyndeo.com](https://app.resyndeo.com) and open **Clusters**, then pick a cluster.

## What's on the page

- **Identity** — cluster name, ID, and registration time.
- **Credentials** — expiry status for the cluster credential.
- **Deployments** — live list of Deployments running on the cluster.

If no deployments have landed yet, the table shows an **Install a plugin** call to action — see [Install a plugin](/install-plugin).

## Prerequisites

- Cluster status is `ready` (the [install flow](/install-cluster) has finished).
- The in-cluster daemon is connected, so deployment state is being projected back to the backend.
